Highlights

  • We propose a new paradigm for continual adaptation of pretrained robot policies: learn residual policies via RL and merge compatible models.
  • We explore model merging in residual RL and improve the merging metric, achieving better success rates after merging.
  • ResMerge achieves both generalization to new task variants and no forgetting of old tasks—addressing the key challenges of continual fine-tuning.

Overview

Problem Formulation

Consider a scenario: you have a generalist robot policy and a robot deployed in a home or factory environment. The robot needs to continuously learn 5–10 new tasks as they arrive sequentially—each task Tk comes with its dataset Dk, and the policy must be adapted to handle all tasks without degrading performance on previously learned ones.

Challenges of Continual Fine-tuning

When continually fine-tuning generalist robot policies, two major challenges arise:

1. Poor Generalization

IL-based PEFT methods often fail to generalize to new task variants (e.g., different object configurations, environmental settings).

2. Catastrophic Forgetting

Traditional fine-tuning approaches cause the model to forget previously learned tasks when adapting to new ones.

Method

ResMerge learns a residual policy Resk for each new task Tk while keeping the base policy frozen. For task Tk+1, we first check if existing policies can generalize; if not, we learn a new residual. We then determine whether the new residual can be merged with existing ones—if yes, we merge into a single model; if no, we add it to the expert pool as an additional specialist.

  • Learn residual RL: Each task is learned as a residual correction on top of the frozen base policy.
  • Model merging: Compatible residuals are merged in parameter space to elicit and combine knowledge.
  • Generalization & no forgetting: Merging can enhance generalization; the design preserves old task performance.

Experimental Setup

We evaluate ResMerge in a continual learning setting where tasks arrive sequentially. For each task Tk, we assess: (1) performance on the single task, (2) generalization to broader task variants, and (3) retention of performance on previously learned tasks.

Results

Main Results

ResMerge learns new skills robustly (high success rate) and generalizes more broadly to new task variants. Model merging performs better with sequential fine-tuning compared to co-finetuning baselines.

Visualization: Before vs. After Merging

Our improved merging metric enables better identification of compatible residuals. Merging compatible models leads to improved success rates and enhanced generalization.

Fusible vs. Non-fusible Groups

Tasks that are similar but have correction action fields with large differences may not be suitable for merging. Our method distinguishes fusible from non-fusible groups to avoid negative interference.

Generalization

ResMerge exhibits strong generalization: when two models both learn some generalization capability for a new task (but insufficient alone), merging can elicit and combine this knowledge, leading to improved performance on the generalized task.
